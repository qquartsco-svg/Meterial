# Rubidium_Cesium_Foundation v0.1.0

루비듐·세슘(Rb/Cs): **1족(알칼리) 중·고원자량 축**으로, `Potassium_Foundation`(K) 다음 단계의 반응성·저장·공급을 스크리닝한다. Na/K와 **동일 취급** 주장을 감점한다.

**정합:** `Potassium_Foundation` → (본 엔진) → 할로겐 층은 `Iodine_Bromine_Foundation` / `Sodium_Chlorine_Foundation` 과 병행.
