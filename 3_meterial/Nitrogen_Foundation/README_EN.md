# Nitrogen_Foundation v0.1.0

**Nitrogen (N₂)** dominates the atmosphere but is kinetically inert; **fixation (Haber–Bosch)** and **air separation** are the engineering pillars.

## Layers

Properties → cryogenic ASU assessment → Haber equilibrium cartoon → LN₂ storage → safety → screening → bridges.

## Limitations

The Haber module is a **cartoon equilibrium**, not a reactor or catalyst model.
