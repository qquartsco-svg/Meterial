# Helium_Foundation v0.1.0

**Helium (He)** is dominated by **sourcing, cryogenics, scarcity, and safety (asphyxiation)** more than reaction chemistry. This package provides an environment to observe that flow.

## Layers

L0 contracts → L1 properties → L2 sourcing (natural gas, not synthesis) → L3 storage → L4 safety → L5 screening → L6 bridges → space (balloon lift helper).

## Quick start

```python
from helium import run_helium_foundation

r = run_helium_foundation()
```

## Limitations

Tree-level cartoons; MRI cryostats and He-3 polarization are future tags.
